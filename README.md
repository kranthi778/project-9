# Network Sniffing and Traffic Analysis

## Overview

This project is about network sniffing and traffic analysis. It shows how to use tools that people in cybersecurity use. The project covers how to capture packets, filter traffic, analyze protocols and do ARP spoofing. We used tools like Bettercap. We looked at how to detect these things from both the guy and the good guy point of view.

We did this project in a lab using Kali Linux, Windows 7 Wireshark, Bettercap and arpspoof.

---

## Objectives

- Learn about network sniffing.

- Use Wireshark to look at captured traffic.

- Apply filters to see packets.

- Look at network conversations and packet details.

- Do ARP Spoofing using arpspoof.

- Do a Man-in-the-Middle attack using Bettercap.

- Check if ARP spoofing is working using Wireshark and ARP cache analysis.

- Learn how SOC analysts detect and stop sniffing attacks.
